## âœ¨ Fitur Utama
1. **Sorting Data Mahasiswa**  
   Mengurutkan berdasarkan nilai (desc), lalu nama (asc) menggunakan `usort()`.

2. **Casting Array Data Orang**  
   Mengubah umur menjadi integer & jenis kelamin jadi "Laki-laki"/"Perempuan".

3. **Sistem Perpustakaan Sederhana**  
   Tambah buku, daftar buku, dan cari buku (terhubung ke database MySQL).

4. **Game Quinque Elementorum**  
   Versi kompleks dari batu-gunting-kertas dengan 5 elemen unik.

5. **Quiz CLI**  
   Kuis interaktif via terminal menggunakan pilihan ganda.

6. **Fungsi Capital Sentence Only**  
   Mengubah string agar hanya huruf pertama kata penting yang kapital.

7. **Penilaian Otomatis Guru**  
   Sistem nilai + kategori remidi & KKM dengan validasi nilai.

8. **Enkripsi & Deskripsi Sederhana**  
   Substitusi huruf sederhana, misal: A = B, M = N.

9. **Pola Segitiga Bilangan Terbalik**  
   Menampilkan segitiga bilangan terbalik dengan nested loop.

10. **Sistem Manajemen Pesan OOP (Multi-Level User)**  
    Pewarisan kelas: User, Moderator, Admin + Logger aksi.

11. **Hitung Bilangan Prima Dalam Rentang**  
    Fungsi `isPrime()` & output jumlah bilangan prima dalam interval.

12. **Hitung Luas Bangun Datar (OOP)**  
    Menggunakan inheritance dan polymorphism (`Persegi`, `Lingkaran`).

13. **Interface Volume Bangun Ruang**  
    Interface `BangunRuang` dengan implementasi `Kubus` & `Bola`.

14. **Fungsi Rekursif Faktorial**  
    Fungsi menghitung faktorial secara rekursif.

15. **Deret Fibonacci**  
    Cetak n-deret Fibonacci menggunakan perulangan.
